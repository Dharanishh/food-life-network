# Food Life Network (FLN)

Food Life Network is a real-time system that connects surplus food from restaurants, hotels, and events with NGOs and hungry communities.

## Problem
Large amounts of edible food are wasted every day while millions of people go hungry because there is no real-time coordination and no trusted food safety system.

## Solution
Food Life Network (FLN) uses Google Forms, Sheets, Maps, and Sites to:
- Collect surplus food in real time  
- Verify food safety using cooking time and QR codes  
- Match NGOs and volunteers  
- Deliver food quickly using Google Maps  

## Technologies Used
- Google Forms  
- Google Sheets  
- Google Maps  
- Google Sites  
- Google Drive  

## MVP
A working MVP built using Google tools that allows donors, NGOs, and volunteers to coordinate in real time.
